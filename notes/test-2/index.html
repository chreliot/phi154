Test 2

(Nov 3)

Test 2 may include the following:

  1. Offer the definition of “logically valid.”
  2. Answer true-false questions using the terms/concepts from the vocabulary list, very similar to the true-false exercises you’ve done previously.
  3. Given a logically valid argument expressed in English, construct a symbolization key for the atomic sentences and/or names and predicates within them, in a way that’s adequate to bring out the logical validity of the argument.
  4. Using a symbolization key, symbolize English sentences in TFL (truth-functional/sentential logic) and FOL (first-order/predicate logic).
  5. Using a symbolization key, translate symbolized TFL and FOL sentences into English.
  6. Construct truth tables for symbolized TFL sentences to show, variously:
    • whether a sentence is a tautology, a contradiction, or contingent;
    • whether a set of sentences is jointly possible/consistent or not;
    • whether an argument is logically valid or not.
  7. For a constructed truth table, write a sentence saying why/how it shows any of the properties mentioned immediately above.
  8. Use the basic and additional natural deduction rules of TFL to prove that a given argument of TFL is valid.
  9. Symbolize an English argument in First-order Logic (FOL).
  10. Use the rules of natural deduction for FOL to prove that a given argument of FOL is valid.
